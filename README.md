# USJ クルーバス時刻表 Web App

USJクルー（スタッフ）専用のバス時刻表PWAアプリです。現在地から次のバスが何分後に来るかを瞬時に確認できます。

## 主な機能

- リアルタイムカウントダウン（1秒更新）
- 次のバスを大きく表示（残り時間 + 発車時刻）
- GPS による最寄り停留所自動検出
- 4つの閉園時間モード対応（19時/20時/21時/22時台）
- 色分け表示（緑: 10分以上、黄: 5-9分、赤: 0-4分、グレー: 発車済）
- オフライン完全対応（Service Worker + LocalStorage）
- PWA対応（ホーム画面追加可能）
- 発車3分前ブラウザ通知
- お気に入り停留所登録（長押し）
- 座標登録モード（管理者用）
- データのエクスポート/インポート

## GitHub Pages で公開する手順

1. GitHubにリポジトリを作成（例: `usj-crew-bus`）
2. ファイルをすべてプッシュ:
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git branch -M main
   git remote add origin https://github.com/YOUR_USERNAME/usj-crew-bus.git
   git push -u origin main
   ```
3. GitHub リポジトリの Settings → Pages へ移動
4. Source を「Deploy from a branch」→ Branch を「main」→ フォルダを「/ (root)」に設定
5. Save をクリック
6. 数分後に `https://YOUR_USERNAME.github.io/usj-crew-bus/` でアクセス可能

## iPhone ホーム画面に追加する手順

1. Safari で公開URLを開く
2. 画面下部の共有ボタン（□に↑）をタップ
3. 「ホーム画面に追加」をタップ
4. 名前を確認して「追加」をタップ
5. ホーム画面にアプリアイコンが追加される

## 使い方

### 基本操作
1. アプリを開く
2. NORTH / SOUTH タブでルートを選択
3. 停留所ボタンをタップして現在地を選択
4. 方向タブ（デンキカフェ行き / クルーサービス棟行き等）を選択
5. 次のバスのカウントダウンが自動表示される

### GPS機能
- 📡 ボタンをタップでGPS取得
- 50m以内: 自動で停留所選択
- 50-150m: 候補として表示（タップで選択）
- 150m超: 手動選択

### お気に入り登録
- 停留所ボタンを長押し（0.8秒）でお気に入り登録/解除
- お気に入りは ⭐ マークで表示

### 閉園時間モード
- 設定画面で手動切替可能
- 「自動取得」選択時はUSJ公式サイトから取得を試行
- 取得失敗時は22時台がデフォルト

### 通知機能
- 設定画面で通知をONにする
- ブラウザの通知許可が必要
- 次のバスの3分前に通知

### 座標登録（管理者用）
- 設定画面の「座標登録」ボタンをタップ
- パスコード: `1234`
- 現在地のGPS座標を各停留所に登録可能

## 技術仕様

- 完全クライアントサイド（HTML + CSS + JavaScript）
- 外部ライブラリ不使用
- PWA (Progressive Web App) 対応
- Service Worker によるオフラインキャッシュ
- LocalStorage によるデータ永続化
- Geolocation API + Haversine公式による距離計算
- Notification API による発車前通知

## 注意事項

- パークオープン 8:30 以降の場合、A便は運休
- GPS座標はサンプル値です。実際の現地で「座標登録」機能を使って正確な座標を登録してください
- 通知機能はブラウザの許可が必要です（iOS Safari では制限あり）
- 時刻表データは2024年時点のものです。変更があった場合は `app.js` 内の `TIMETABLE` オブジェクトを更新してください
